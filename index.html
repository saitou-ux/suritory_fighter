スリトリーファイター

4キャラの選択画面とスプライトシートで構成した、90年代アーケード風の対戦プロトタイプです。 地上のパンチ・キック、ジャンプ攻撃に加えて、波動拳コマンドも試せます。

メニュー

操作方法

タイトル開始: Enter または Z

選択画面: ↑ ↓ ← → で移動

決定 / 試合開始: Enter または Z

戻る / タイトルへ戻る: X

相手決定後: CPU強さを選択

移動: ← →

しゃがみ: ↓

ジャンプ: ↑

パンチ: Z / しゃがみパンチ: ↓ + Z

キック: X / しゃがみキック: ↓ + X

波動拳: ↓ ↘ → + Z

ガード: 相手と反対方向を入力

全画面: F

戦闘システム

今回は4キャラ選択とスプライト表示を優先した簡略版です。

使用アクションは立ち、歩き、しゃがみ、ジャンプ、パンチ、キック、波動拳、ガードです。

ジャンプ中に Z / X を押すとジャンプ攻撃、しゃがみ中に Z / X を押すとしゃがみ攻撃が出せます。

↓ ↘ → + Z で波動拳が出せます。CPU も距離を取りながら波動拳と近接を使い分けます。

地上で後ろ入力をしている間に攻撃を受けると、立ちガードまたはしゃがみガードが発動します。

相手決定後は CPU の強さを「やさしい / ふつう / 最強」から選べます。

効果音は最初のキー入力またはボタン操作後に有効化され、メニュー、攻撃、ガード、被弾、波動拳、ラウンド演出が鳴ります。

先に2ラウンド取った側が勝利です。

現在の段階

メニュー操作音、攻撃音、ガード音、被弾音、波動拳音、ラウンド開始と決着演出の効果音まで入っています。
